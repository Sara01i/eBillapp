# eBill App

تطبيق محفظة ذكي لإدارة الفواتير مصمم بـ Flutter و Firebase.

## المميزات
- تسجيل دخول وإنشاء حساب عبر البريد الإلكتروني.
- نظام تحقق (OTP) آمن عبر البريد الإلكتروني باستخدام Node.js و Gmail SMTP.
- إدارة الملف الشخصي وتحديث البيانات.
- واجهة مستخدم داكنة (Dark Theme) حديثة وأنيقة.

## المتطلبات
- **Flutter SDK**: الإصدار 3.11.0 أو أحدث.
- **Node.js**: للإشراف على خادم التحقق (Backend).
- **Firebase Project**: لإدارة قاعدة البيانات والهوية.

## طريقة التشغيل

### 1. إعداد خادم التحقق (Node.js)
1. انتقل لمجلد السيرفر:
   ```sh
   cd server
   ```
2. قم بتثبيت المكتبات:
   ```sh
   npm install
   ```
3. قم بإعداد ملف `.env` بناءً على `.env.example`.
4. ضع ملف `serviceAccountKey.json` الخاص بـ Firebase داخل مجلد `server`.
5. قم بتشغيل السيرفر:
   ```sh
   node index.js
   ```

### 2. تشغيل تطبيق Flutter
1. تأكد من أن هاتفك أو المحاكي متصل.
2. قم بتحميل مكتبات Flutter:
   ```sh
   flutter pub get
   ```
3. تشغيل التطبيق:
   ```sh
   flutter run
   ```

## ملاحظات هامة
- عند التشغيل على محاكي الأندرويد، يتصل التطبيق بالسيرفر عبر `10.0.2.2:3000`.
- تأكد من تفعيل "App Passwords" في حساب Gmail الخاص بك لاستخدامه في إرسال الـ OTP.
